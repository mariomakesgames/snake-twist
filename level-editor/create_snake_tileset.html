🐍 Snake Tileset Generator

为Tiled地图编辑器生成 snake_tileset.png 文件

使用说明：
  1. 点击"生成图块集"创建图像
  2. 点击"下载 PNG"保存为 snake_tileset.png
  3. 将PNG文件放在与Tiled地图文件相同的目录
  4. 在Tiled中打开 tiled_snake_level.json

图块说明

ID 1: 背景 (深灰)
ID 2: 空地 (白色)
ID 3: 障碍物 (灰色)
ID 4: 普通食物 (绿色)
ID 5: 成长食物 (黄色)
ID 6: 速度食物 (橙色)
ID 7: 缩短食物 (红色)
ID 8: 减速食物 (粉色)
ID 9: 传送门 (蓝色)
ID 10: 蛇出生点 (青绿)
